# 添加功能

需要在已有的功能上添加新的功能：

添加降采样+bootstrap功能。

考虑到遗传学分析中不同群体之间的样本量差异可能会影响结果的可靠性，降采样和bootstrap方法可以帮助我们评估结果的稳定性和可靠性。

## 降采样策略

原则

每次纳入的样本量 n，取所有group中可用样本数的最小值。

例如样本数分别是：

A = 18
B = 42
C = 67

统一降采样量就是：

n = 18

## Bootstrap策略

默认设置为1000次bootstrap。

每次bootstrap时，随机从每个群体中抽取n个样本（有放回），计算所需的统计量，并记录结果。
输出时报告：

1. 平均值
2. 标准差
3. 2.5 百分位
4. 50 百分位
5. 97.5 百分位

## bootstrap 是对样本抽（不是对位点抽！）

样本 bootstrap：对样本进行有放回抽样，多次计算`/mnt/c/Users/Administrator/Desktop/scikit-allel/python`中的统计量，评估结果的稳定性和可靠性。

mtDNA 是一个很短的连锁分子，位点并不独立。所以对 mtDNA，**不建议**逐位点 bootstrap。

遵循目前已有的文件组织结构。
