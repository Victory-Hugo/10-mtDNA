# phyloMT 使用教程

## 1. 当前正确使用方式

当前 `phyloMT` **不是打包软件**，也**不是全局命令**。

请直接在项目根目录下使用：

```bash
python3 phyloMT.py --tree ... --input ... --output ...
```

推荐完整写法：

```bash
/home/luolintao/miniconda3/envs/BigLin/bin/python phyloMT.py ...
```

先进入项目目录：

```bash
cd /home/luolintao/P1-LLT-23K-mtDNA/10-单倍群分型软件/3-phyloMT
```

## 5. 核心分类命令

通用格式：

```bash
/home/luolintao/miniconda3/envs/BigLin/bin/python phyloMT.py \
  --tree <树ID> \
  --input <输入文件> \
  --output <输出文件>
```

当前支持输入格式：

- `HSD`
- `VCF`
- `FASTA`

## 9. 常用参数

当前接口支持：

- `--tree`：树 ID，例如 `phylotree-fu-rcrs@1.2`、`phylotree-rcrs@17.2`
- `--input`：输入文件
- `--output`：输出文件
- `--hits`：输出前 N 个候选
- `--het-level`：VCF 杂合阈值
- `--extended-report`：输出扩展报告
- `--write-fasta`：额外输出重建 FASTA
- `--trees-dir`：手动指定树目录
- `--list-trees`：列出当前树目录下全部可用树

## 10. 输出文件说明

### 11.1 主输出

主输出是一个制表符分隔文件，主要列为：

- `SampleID`
- `Haplogroup`
- `Rank`
- `Quality`
- `Range`

例如：

```text
"SampleID"	"Haplogroup"	"Rank"	"Quality"	"Range"
"Sample1"	"H100"	"1"	"0.8356"	"1-16569"
```

### 11.2 扩展报告

如果加：

```bash
--extended-report
```

会额外生成：

```text
<输出文件名>.extended.txt
```

例如：

```text
output/H100.extended.txt
```

扩展列包括：

- `Input_Sample`
- `Found_Polymorphisms`
- `Missing_Polymorphisms`
- `Extra_Polymorphisms`
- `Found_Weight`
- `Expected_Weight`
- `Sample_Weight`

### 11.3 重建 FASTA

如果加：

```bash
--write-fasta
```

会额外生成同名前缀的 `.fasta` 文件。

例如：

```bash
--output output/H100.txt
```

会生成：

```text
output/H100.fasta
```
