## 1. Complete sequence

适用位点：

```text
所有 substitutions，但排除 16519、16182C、16183C、16194C
```

定义函数：

```text
F(x) = 3624 × x × 2.71828^[-2.71828^((x + 40.2789) × -0.0263)]
```

则：

```text
age = F(ρ)
ci95_lower = F(ρ - 1.96 × SE)
ci95_upper = F(ρ + 1.96 × SE)
```

也就是：

```text
age =
3624 × ρ × 2.71828^[-2.71828^((ρ + 40.2789) × -0.0263)]
```

```text
ci95_lower =
3624 × (ρ - 1.96 × SE) × 2.71828^[-2.71828^(((ρ - 1.96 × SE) + 40.2789) × -0.0263)]
```

```text
ci95_upper =
3624 × (ρ + 1.96 × SE) × 2.71828^[-2.71828^(((ρ + 1.96 × SE) + 40.2789) × -0.0263)]
```

## 程序实现版本

```python
def complete_sequence_age(rho):
    return 3624 * rho * (2.71828 ** (-(2.71828 ** ((rho + 40.2789) * -0.0263))))

def linear_age(rho, se, years_per_mutation):
    return {
        "age": rho * years_per_mutation,
        "ci95_lower": (rho - 1.96 * se) * years_per_mutation,
        "ci95_upper": (rho + 1.96 * se) * years_per_mutation,
    }

def soares_complete(rho, se):
    return {
        "age": complete_sequence_age(rho),
        "ci95_lower": complete_sequence_age(rho - 1.96 * se),
        "ci95_upper": complete_sequence_age(rho + 1.96 * se),
    }

def soares_synonymous(rho, se):
    return linear_age(rho, se, 7872)

def soares_hvsi_16051_16400(rho, se):
    return linear_age(rho, se, 16677)

def soares_hvsi_16090_16365_transitions(rho, se):
    return linear_age(rho, se, 18845)

def soares_hvsii_68_263(rho, se):
    return linear_age(rho, se, 22388)

def soares_control_region(rho, se):
    return linear_age(rho, se, 9058)
```

注意：这些公式**不自动把负的 lower bound 截断为 0**。如果 `ρ - 1.96 × SE < 0`，严格复现 Excel 时仍按原公式计算。
