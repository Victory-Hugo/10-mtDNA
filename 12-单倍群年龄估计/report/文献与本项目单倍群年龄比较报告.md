# 文献单倍群年龄与本项目年龄估计比较报告

## 1. 数据来源与处理

- 文献输入目录：`download/`，共解析 15180 条文献年龄记录，覆盖 7392 个唯一单倍群。
- 本项目输入：`output/stage_2/results/ml_vs_rho_comparison.tsv`，共 6234 个单倍群。
- 所有年龄统一换算为 kya；`以往研究-1.tsv` 与 `以往研究-4.tsv` 原始单位为 years，已除以 1000。
- 对只有区间的 ML 年龄，使用区间中点作为文献年龄，同时保留上下界；对仅给出 SD 的表，使用 age ± 1.96 SD 近似 95% CI。
- 合理性主判定采用区间覆盖规则：文献年龄上下限与本项目 ρ 或 ML 的 95% CI 只要有任一交集，即认定为合理。

## 2. 总体一致性

- 与本项目存在同名单倍群匹配的文献记录：7806 条。
- 与本项目存在同名单倍群匹配的唯一单倍群：3638 个。
- 文献有记录但本项目未匹配的记录：7374 条。
- 可进行区间覆盖判定的记录：7290 条。
- 区间与本项目 ρ 或 ML 至少一个结果有覆盖、因此认定合理的记录：6476 条。
- 区间与本项目 ρ 和 ML 均不覆盖的记录：814 条。
- 与本项目 ρ 年龄几乎相同（绝对差异 < 0.01 kya）的记录：89 条。
- 与本项目 ML 年龄几乎相同（绝对差异 < 0.01 kya）的记录：85 条。
- 绝对差异和相对差异仍保留在结果表中，但只作为参考指标，不作为合理性主判定。
